Это простой Laravel-проект с авторизацией (регистрация/вход) и одной защищённой страницей, на которой отображаются данные авторизованного пользователя: ID, имя и email.

Как запустить проект
1. Установка зависимостей 
composer install
npm install
2. Настройка окружения

Если файла .env нет — создайте:
cp .env.example .env\
Сгенерируйте ключ приложения:
php artisan key:generate
Укажите настройки базы данных в .env.
3. Миграции
php artisan migrate
4. Запуск сервера
 php artisan serve
 Регистрация и вход

Страницы доступны сразу после установки Breeze:

Регистрация: /register

Вход: /login

Создайте аккаунт, чтобы войти.

Защищённая страница профиля

После входа перейдите на:

/profile


Там выводятся:

ID пользователя

Имя

Email
